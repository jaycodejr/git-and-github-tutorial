GIT AND GITHUB CONFIGURATION STEPS
1. Configure git on PC
    i. Open terminal and run the following commands
    - git config --global user.name "<your username>"
    - git config --global user.email "<your email address>"

2. How to create a project and push to github
    i. Create a repository on your github account
    ii. Create a project and navigate to the project folder
    iii. Run the following commands
    - git init
        -> tells git to start tracking the project
    - git branch -M main
        -> changes master name branch to main
    - git remote add origin "<remote repo url>"
        -> link remote repo to local PC
    - git push -u origin main
        -> pushes local project to the remote repository

3. Create new branch
    i. Open terminal line and run the following command
    - git checkout -b "<branch name>"
        -> creates new branch and switches to the new created branch

4. Merge two branches
    i. Open terminal and run the following command
    - git merge "<branchA>"
        -> merges current branch with branchA
